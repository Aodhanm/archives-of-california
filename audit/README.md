## Rubric (per record, against the manuscript leaf)
- **DATE**: OK / ERR (wrong day, month, or year) / PART (imprecise but not wrong, e.g. month-only where leaf has day) / UV (unverifiable at leaf)
- **PERSONS**: author + recipient as recorded — OK / ERR / PART / UV
- **SUBSTANCE**: summary faithfully reflects the document — OK / ERR (misrepresents) / PART (accurate but materially incomplete) / UV
- **PIN**: page pins (orig_page / savage_page / scan) point at the document — OK / ERR / PART
- One line of notes; every ERR also goes to the errata register with its class.

## Verdicts
Accumulate in `verdicts.csv` (n, ca, doc, date, persons, substance, pin, note). Sessions append; the audit is complete when all 200 rows are graded. Published rate = errors/graded by field, stated with sample size and date on the About tab.
