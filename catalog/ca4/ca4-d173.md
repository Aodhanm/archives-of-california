# Baja indigenous-agriculture + supply report (via González): describes damming an arroyo for a maize planting ('máis chiq

**C-A 4 (Provincial State Papers), scan n175** · Sav p. 175–177 · tomo p. 552–556 · 1787-04-05 · Savage abstract · 

Baja indigenous-agriculture + supply report (via González): describes damming an arroyo for a maize planting ('máis chiquito del Colorado'), temporal wheat; + Ortega 'Viaje al canal. Modo de avaluar unas mulas. Sorteo de caballos' — promises to be at the Canal by 15 June per superior order; Fr. García to draw 2 horses by lot. Orig. pp.552–556.
