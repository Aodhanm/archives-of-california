# ⚠ BOUNDED ROUTINE RUN (per efficiency rule 6): a long stretch of printed general-empire royal cédulas transmitted by de 

**C-A 6 (Provincial State Papers), scan n96** · Sav p. 95–118 · tomo p. ~325–419 · 1789–1791 · Savage abstract · 

⚠ BOUNDED ROUTINE RUN (per efficiency rule 6): a long stretch of printed general-empire royal cédulas transmitted by de Nava — zero California specificity, so noted as a block rather than leaf-by-leaf. Sampled contents: ministers/Real-patronato provision + a Durango/Sonora *doctrinas* dictamen (n100); ⭐ indulto general del delito de contrabando (a Viceroy's bando pardoning the contraband crime for non-homicides, n105); requirements for títulos de Castilla / hidalguía (n110); ⭐ the Bourbon cemetery-reform cédula (burials out of churches for public health, from Gov. Ezpeleta of Cuba's 1787 report, n115). ⏳ If item-level completeness is ever needed, read n96–n119 leaf-by-leaf; for now characterized + bounded.
