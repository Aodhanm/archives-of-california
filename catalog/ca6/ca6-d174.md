# Sal's SECOND SF-presidio report: the $1,200 construction gratification never arrived (vs $4,000 to each Provincias-Inter

**C-A 6 (Provincial State Papers), scan n237–n242** · Sav p. XI:53–58 · tomo p. 178–183 · 1792-12-29 · Savage abstract · 

Sal's SECOND SF-presidio report: the $1,200 construction gratification never arrived (vs $4,000 to each Provincias-Internas presidio — Sal's pointed complaint); no worker list; the construction history again; the lienzos building-by-building — almacenes/guardia/cuartel 'viniendo abajo,' commandant's & troop houses needing repair, the church to be rebuilt for lack of foundations, the zacate/tule roofs fire-exposed, the 4th lienzo mere palisade; only ~6 usable houses; asks for 8–10 servants/San Blas sailors + a master mason for 4 yrs OR 3,000 pesos to keep building. ⚠️ n240 (the roof-repair/wind/*tule-3-leguas*/few-forces/supply-ship passage) is PART OF THIS SF REPORT — see the audit re-correction note below
