# ⚠ BOUNDED 1792 cédula-duplicate run (re-published by Romeu Jan 1792 + Argüello/the 5 commanders Jul–Oct 1792): Castilla 

**C-A 6 (Provincial State Papers), scan n213–n218** · Sav p. XI:29–34 / 112–144 · 1791–92 (re-pub.) · Savage abstract · 

⚠ BOUNDED 1792 cédula-duplicate run (re-published by Romeu Jan 1792 + Argüello/the 5 commanders Jul–Oct 1792): Castilla titles, concubinato, the matrimonios pragmática (Lebrun), Marina-officer declarations, the slave-*alcabala*/manumission cédula, confraternity juntas, the papal *meat-on-Saturdays* indult (Pius VI), ministers' obligations — all DUPs of the Tomo-X de Nava cédulas (the slave-manumission + meat-indult are the only new ones). + a minor Sal soldier-return note (n213).
