# a 4-article protocol for the commissioners accompanying the departed New-Mexican traders' party: on reaching the New Mex

**C-A 34 (Departmental State Papers, Los Angeles), scan n500–n501** · Sav p.  · tomo p. 500–501 · 1840-04-08 · Savage abstract · 

a 4-article protocol for the commissioners accompanying the departed New-Mexican traders' party: on reaching the New Mexicans, state the commission's object; with the comandante, inspect each owner's animals, list them & seize any ill-gotten (*mal habidas*); report those traveling without passports; on completion give the NM comandante a certificate of the herd's legality & file a list — see E28 (doc pp.278-282)
