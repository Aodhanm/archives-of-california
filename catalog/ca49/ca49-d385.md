# the Hacienda in very bad state — blames Echeandía's administration: the useful cattle-ranchos finished off, foreign ship

the Hacienda in very bad state — blames Echeandía's administration: the useful cattle-ranchos finished off, foreign ships paying in goods via Echeandía-placed employees (Bandini, José Antonio Estudillo), unsecured customs houses acting 'absolutos' (Bandini stood guarantor for Fr.
